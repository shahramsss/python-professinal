List Comprehension

[expression for item in iterable if condition]

expression: عملی که روی هر آیتم انجام می‌دی (مثلاً مربع کردن یا تبدیل به رشته بزرگ)
item: هر عنصر از مجموعه
iterable: مجموعه‌ای که روی آن پیمایش می‌کنی (مثل range یا لیست)
condition: شرط اختیاری برای فیلتر کردن عناصر


squares = [x**2 for x in range(1, 6)]
# خروجی: [1, 4, 9, 16, 25]

evens = [x for x in range(10) if x % 2 == 0]
# خروجی: [0, 2, 4, 6, 8]

names = ["ali", "sara", "reza"]
upper_names = [name.upper() for name in names]
# خروجی: ['ALI', 'SARA', 'REZA']

words = ["apple", "banana", "cherry"]
lengths = [len(word) for word in words]
# خروجی: [5, 6, 6]
